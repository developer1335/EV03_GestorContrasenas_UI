GESTOR DE CONTRASEÑAS (Prototipo UI con TailwindCSS)
---------------------------------------------------
Este proyecto consiste en el desarrollo de un prototipo de interfaz de usuario (UI) para un Gestor de Contraseñas, elaborado con HTML, JavaScript y TailwindCSS.
Su objetivo es proporcionar una base sencilla pero escalable para la gestión de contraseñas de forma segura y organizada.

📂 Estructura del proyecto
EV03_GestorContrasenas_UI/
│── index.html      # Estructura principal de la aplicación
│── app.js          # Lógica básica en JavaScript
│── README.md       # Documentación del proyecto

Funciones implementadas:
- Registro de usuarios.
- Inicio de sesión.
- Guardar contraseñas por servicio.
- Ver todas las contraseñas guardadas.
- Buscar por servicio.
- Eliminar contraseñas.
- Cerrar sesión.
- Salir de la aplicación.

Tecnologías usadas:

- HTML5 → Estructura del contenido.
- TailwindCSS → Estilos modernos y responsivos.
- JavaScript (Vanilla JS) → Lógica y funcionalidades básicas.
--------------------------------

Instrucciones:
1. Abre "index.html" en tu navegador.
2. Regístrate o inicia sesión.
3. Usa el menú de usuario para administrar contraseñas.
4. Cerrar sesión para volver al menú principal.


Clona este repositorio o descarga los archivos.
